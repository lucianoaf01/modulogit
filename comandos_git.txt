
git add -A -> adciona no arquivo

git commit -am "teste" ->   cria um commit salvando as alterações

git branch ->   mostra o branch atual

git branch teste -> cria um novo branch

git checkout teste -> muda para o branch teste

git diff ->    mostra as alteraçoes feitas

git checkout HEAD -- style.css ->  retorna ao estado original do arquivo

git diff --name-only ->  mostra os arquivos modificados

git remote -> mostra a origem

git remote -v -> mostra opções fetch e push

git push -u origin master -> envia arquivos para o git remoto

git push origin master -> envia arquivos para o git remoto

.gitignore -> serve para ignorar 

git push origin :teste -> deleta um branch remoto

git branch -D teste -> deleta um branch local
